# Guide media

Drop screenshots and short clips here. The `GuidePage` references them by
filename — when a file is missing, a dashed-border placeholder renders in
its place with the filename + intended caption so it's obvious what's
still to capture.

## Filename convention

`NN-short-description.{png,mp4}` — two-digit prefix groups the sections in
file-explorer order, hyphenated kebab-case for the description, png for
stills, mp4 for clips.
